## User Guide
First, Create arduino_secrets.h. \
Second, Define SSID and PASSWORD.
```c
#define SECRET_SSID "..."
#define SECRET_PASS "..."
```
Third, Download latest "ArduinoJson" library using library manager in arduino IDE.
